 Tienda de Relojes - JavaScript Dinámico

E-commerce completo desarrollado con JavaScript vanilla, con carrito de compras dinámico y persistencia de datos.


[Ver sitio publicado](https://sergio-centurion.github.io/tienda-dinamico.js/)

 Características
 Catálogo dinámico desde JavaScript
 Carrito funcional con localStorage
 Diseño 100% responsive
 Navegación entre páginas
 Formulario de contacto funcional

Tecnologías
- HTML5, CSS3, JavaScript ES6+
- localStorage para persistencia
- CSS Grid & Flexbox
- Google Fonts
